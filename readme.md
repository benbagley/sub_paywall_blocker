# Sub Paywall Blocker

This extension will bypass news subscription paywalls.

## Getting Started

### Paywalls supported

This extension currently supports the following websites:

- dailytelegraph.com.au
- telegraph.co.uk
- ft.com
- suzylu.co.uk

### Build

- Clone
- cd into the folder
- Run `npm install --global web-ext`
- Run `web-ext run` (install web-ext first)
